### Troubleshooting:
**Favicon not showing?**
- Clear browser cache (Ctrl+Shift+Delete)
- Hard reload (Ctrl+Shift+R)
- Check `/public/icon.svg` exists
- Restart dev server

**Social preview not showing?**
- Must be deployed to public URL (not localhost)
- Use validators to force refresh
- Facebook caches - use debugger to scrape again
- Twitter may take 5-10 minutes to update
